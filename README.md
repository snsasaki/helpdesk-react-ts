# ヘルプデスク

## 概要

- 社内問い合わせ管理アプリ
- react-tsの学習用リポジトリ

## 機能一覧

### 必須機能

- 問い合わせ一覧
- 詳細表示
- 新規登録
- ステータス更新
- 画面切り替え

### 追加機能

- ステータスでフィルタ
- バリデーション
- スタイリング

### チャレンジ機能

- 並び替え
- 削除機能
- 件数表示

## 技術スタック

- TypeScript
- React
- CSS
- Material UI
